# TradeMcpNet v1.0.0

Initial release of TradeMcpNet - an MCP (Model Context Protocol) server for E*TRADE API integration with AI agents.

## Features

### MCP Server
- Full MCP server implementation using stdio transport
- Compatible with Claude Desktop, VS Code with GitHub Copilot, and other MCP clients

### OAuth 1.0a Authentication
- Interactive OAuth flow designed for AI agent workflows
- `etrade_oauth_start` - Initiate authentication and get authorization URL
- `etrade_oauth_complete` - Complete authentication with verifier code
- `etrade_oauth_status` - Check current authentication status
- `etrade_oauth_renew` - Renew access token before expiration
- `etrade_oauth_revoke` - Log out and revoke access token

### E*TRADE API Tools
- Auto-generated tools from E*TRADE's OpenAPI specification
- Account management operations
- Portfolio and position queries
- Order placement and management
- Market data and quotes
- Option chains and alerts

### Configuration
- Environment variable configuration support
- Command line argument support
- Sandbox environment support for safe testing

### Distribution
- Available as a .NET global tool (`etrade-mcp`)
- Supports .NET 8.0 and .NET 9.0

## Installation

```bash
dotnet tool install --global TradeMcpNet.Server
```

## Getting Started

1. Obtain API credentials from the [E*TRADE Developer Portal](https://developer.etrade.com/)
2. Configure your MCP client with the `etrade-mcp` command
3. Use the OAuth tools to authenticate
4. Start interacting with E*TRADE through your AI assistant

For detailed usage instructions, see the [README](https://github.com/kerryjiang/TradeMcpNet).
